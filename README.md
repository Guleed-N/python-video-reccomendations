# YouTube Recommendation Email Script

This Python script uses the YouTube API to fetch video recommendations based on user input and sends them to an email address using the Gmail API.

## Prerequisites

Before running the script, ensure you have the following:

1. **YouTube API Key:**
   - Obtain a YouTube API Key from the [Google Cloud Console](https://console.cloud.google.com/apis/credentials).
   
2. **Gmail API Credentials:**
   - Create a service account on the Google Cloud Console
   - Download the JSON key file for the service account and save it in the project directory.
